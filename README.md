# retail-profitability-analysis
Retail Profitability & Pricing Strategy Analysis
About This Project

This project started as a simple question:
Why is the Furniture category so much less profitable than the others?

I used SQL Server to clean and analyze the data, then built a Power BI dashboard to better understand what was driving the low margins and to test what would happen if certain pricing changes were made.

What I Found

Overall profit margin: 12.47%

Furniture margin: around 2.5%, much lower than other categories

Discounts of 30% or more were generating about $54K in negative profit

Removing those high-discount transactions increased projected margin to 16.22%

The main takeaway: deep discounting in Furniture was heavily impacting profitability.

Tools I Used

SQL Server – data cleaning, KPI calculations, and scenario modeling

Power BI – dashboard creation and visualization
